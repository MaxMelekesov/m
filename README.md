# m
Библиотека для написания кода на C++ под микроконтроллеры.

Предоставляет интерфейсы, примеры реализации драйверов для периферии МК, классы для внешних микросхем и часто используемые готовые логические блоки.

Не используется: выделение памяти, RTTI, исключения

Приоритеты: максимум читаемости и поддерживаемости кода
